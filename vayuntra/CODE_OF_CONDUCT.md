# Code of Conduct

## Our Pledge

We are committed to maintaining a professional, inclusive, and respectful environment for all contributors.

## Standards

- Use professional and respectful language
- Provide constructive feedback
- Respect differing viewpoints
- Focus on what is best for the project and its users

## Enforcement

Violations should be reported to: conduct@vayuntra.internal

Maintainers reserve the right to remove contributions that violate this Code of Conduct.

## Scope

This Code of Conduct applies to all project spaces including GitHub Issues, Pull Requests, and Discussions.
